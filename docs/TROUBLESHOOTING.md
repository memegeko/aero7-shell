# Troubleshooting

Start with:

```bash
aero7 doctor
aero7 logs
```

Common recovery commands:

```bash
aero7 backups
aero7 restore --latest
aero7 repair
aero7 plymouth status
aero7 plymouth repair
```

Browser theming is intentionally excluded because earlier browser theme experiments caused browser glitches. Aero7-shell does not install Geckium, Aero UserChrome, or any browser profile modifications.
