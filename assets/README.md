# Aero7-shell Assets

Aero7-shell only distributes assets whose origin and licence are recorded in `docs/ASSET-LICENSING.md`.

Current distributed assets:

- `assets/avatars/aero7-user.png`
- `assets/fastfetch/aero7.jsonc`

No wallpaper is currently approved for distribution. The installer skips wallpaper application when no approved wallpaper exists.

Do not add Microsoft wallpapers, Windows logos, usertile BMP files, extracted DLL/EXE resources, Microsoft sounds, Microsoft icons, or Microsoft fonts to this directory.
